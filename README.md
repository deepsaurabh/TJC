# Skeleton with React Redux Saga

[![Greenkeeper badge](https://badges.greenkeeper.io/mCodex/react-redux-saga-boilerplate.svg)](https://greenkeeper.io/)

## Features Out-Of-The-Box

* React-Router 4
* Semantic Ui
* Redux
* Redux Saga
* ESlint
* Airbnb's ESlint rules
## Installation

Clone repo and run:

```
npm i && npm start
```
